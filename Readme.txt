# Grammar Checker Web App

A lightweight **AI-powered Grammar Checker** web application built using **Flask**, **NLTK**, and **LanguageTool**, designed to identify and correct grammatical errors in real-time.

## Overview

This application helps users improve their writing by highlighting grammar issues and suggesting corrections. It combines traditional grammar-checking tools with a custom **rule-based AI agent** that evaluates grammar using natural language processing techniques.

## Features

- Real-time grammar checking
- Custom-built AI agent for grammar rule analysis
- Rule-based NLP processing using NLTK
- Integrated with LanguageTool for advanced grammar detection
- Context-aware feedback for tense, syntax, punctuation, and usage
- Clean UI with error highlighting and suggestions
- Flask-based backend for smooth integration and performance

## Technologies Used

- Python
- Flask
- NLTK (Natural Language Toolkit)
- LanguageTool Python API
- HTML/CSS (for frontend)
